# Projeto Média em Python 💻

Esse projeto foi desenvolvido em Python para calcular automaticamente a média de notas dos alunos. O sistema permite inserir as notas, realizar o cálculo da média final e informar se o aluno foi aprovado ou reprovado com base na média estabelecida.

Durante o desenvolvimento, foram utilizados conceitos de lógica de programação, variáveis, operadores matemáticos, estruturas condicionais e entrada de dados no terminal, proporcionando prática no desenvolvimento de aplicações simples e funcionais. 

## Sistema de Média de Notas ⚙️
Projeto desenvolvido para calcular a média de alunos de forma automática, permitindo inserir notas, realizar cálculos e informar o resultado final. O sistema utiliza estruturas condicionais para verificar aprovação ou reprovação, além de trabalhar com entrada e saída de dados no terminal.

## Conclusão 🚀
O desenvolvimento do projeto de média de notas em Python foi importante para aprimorar conhecimentos em lógica de programação e manipulação de dados. Através dele, foi possível praticar cálculos matemáticos, estruturas condicionais e interação com o usuário, além de compreender melhor a criação de sistemas simples e funcionais. O projeto contribuiu para o desenvolvimento de habilidades essenciais na programação e resolução de problemas.
